This folder contains the datasets and code used for the analysis of forest carbon projects.

## Files

### 1. `carbonplan_merged_forest_verra`
Transaction-level CarbonPlan data for Verra forest projects.  
This dataset includes detailed transaction records, including transaction dates and transaction notes.

### 2. `full_list_107.xlsx`
The final list of 107 projects included in the scoring analysis.  
This file contains project-level scores for:
- GHG
- Co-benefits
- SG dimensions

### 3. `Code`
This folder contains the code used for the data analysis.  
The analysis uses both datasets above.

## Notes
- `carbonplan_merged_forest_verra` provides the transaction-level market data.
- `full_list_107.xlsx` provides the final scored sample used in the empirical analysis.
- The code links and analyzes information from both datasets.
